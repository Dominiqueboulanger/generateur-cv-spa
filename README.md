# Générateur de CV SPE

Ce projet est un outil interactif permettant aux professionnels du secteur du Salarié du Particulier Employeur de générer un CV au format PDF/A4 de manière simple et structurée.

## 🚀 Fonctionnalités
- **Référentiel métier :** Chargement automatique des compétences techniques et humaines via un fichier JSON.
- **Édition dynamique :** Saisie des expériences, formations et informations personnelles avec aperçu en temps réel.
- **Export/Import :** Possibilité de sauvegarder et charger ses données via des fichiers `.json`.
- **Mise en page A4 :** Optimisé pour l'impression ou la sauvegarde en PDF.

## 🛠️ Installation et Utilisation
Aucune installation n'est requise. Il suffit d'ouvrir le fichier `Generateur_CV_SPE.html` dans un navigateur moderne ou d'utiliser le lien GitHub Pages associé.

## 📁 Structure du projet
- `Generateur_CV_SPE.html` : L'application principale (HTML/CSS/JS).
- `competences_SPE_par_métier.json` : La base de données des compétences par métier.
